# Task 4.0 — Implementar temas mensais e frases do dia

- **Complexidade:** MEDIUM
- **Dependências:** 1.0
- **Status:** completed

## Objetivo
Adicionar a lógica mínima para temas mensais estoicos e frases do dia no onboarding/introduction, alinhando o app ao conteúdo descrito no PRD e na spec.

## Contexto PRD
A especificação técnica apresenta 12 lições mensais estoicas e frases diárias multilíngues. A primeira versão deve disponibilizar uma estrutura para exibir temas e frases de forma simples.

## Requisitos Técnicos
- Criar uma estrutura de dados para temas mensais.
- Exibir uma frase do dia no onboarding ou intro.
- Permitir a seleção de idioma inicial (implementação mínima se necessário).
- Manter o layout atual intacto.

## Critérios de Aceitação
- O app exibe uma frase estoica relevante na tela de abertura ou intro.
- A frase e o tema do mês podem ser alterados pelo estado interno.
- A implementação não altera o layout atual.

## Subtarefas
1. Criar dados iniciais de frases e temas mensais.
2. Conectar os dados à tela de onboarding/intro.
3. Adicionar fallback caso não haja língua selecionada.

## Testes Planejados
- Teste de unidade: função utilitária que seleciona a frase do dia.
- Teste de integração: garantir renderização da frase no onboarding.

## Resultado
- Criada estrutura de temas mensais em `src/lib/stoicThemes.ts`.
- Onboarding agora usa o tema do mês e a frase do dia dinamicamente.
- A seleção de idioma é inferida do navegador com fallback para `pt-BR`.
- O layout do onboarding permaneceu intacto.
